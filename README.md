# QCAAS - Full Stack Project

A professional web application integrating a Flask backend with a React (Vite) frontend.

---

## Directory Structure

* **`backend/`**: Python Flask server, environment configuration, and ML logic.
* **`client/`**: React frontend built with Vite and Tailwind CSS.

---

## Getting Started

### 1. Backend Setup (Flask)

Open your VS Code terminal (**Ctrl + `**) and run the following:

```powershell
# Navigate to backend
cd backend

# Create a virtual environment
python -m venv venv

# Activate the virtual environment (Windows)
.\venv\Scripts\activate

# Install required packages
pip install -r requirements.txt

# Start the Flask server
python app.py
